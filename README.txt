== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate TP Advanced Search For WooCommerce through the 'Plugins' menu in WordPress.

TP Advanced Search For WooCommerce will add a new menu item called “Advanced Search” in wordpress menu. Here you are able to configure all the plugin settings.
